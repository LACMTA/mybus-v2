Metro вносит ряд изменений в график движения автобусов.

С воскресенья, 23 октября, вступают в силу изменения в графике движения некоторых автобусных маршрутов Metro. Это позволит сделать наши перевозки еще более надежными и обеспечить их пунктуальность.

По этим маршрутам изменится частота движения.

Будни: 4, 18, 30, 53, 66, 70, 90, 92, 180, 207, 210, 212, 217, 224, 240, 690

Суббота и воскресенье: 2, 4, 28, 210, 224, 690

По этим маршрутам будут добавлены дополнительные рейсы для увеличения пропускной способности.

Будни: 108, 224, 460, 534

Суббота: 33, 212

Воскресенье: 212
